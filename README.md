## Описание задачи

Разработка новых лекарственных препаратов - сложный и длительный процесс, включающий синтез соединений и их биологическое тестирование. Современные методы машинного обучения позволяют ускорить этот этап, предсказывая эффектиновсть химических соединений до проведения лабораторных экспериментов.

Химики предоставили данные о свойствах молекул и их биологической активности против вируса гриппа. 
**Основная задача** - построить модели, способные предсказывать эффективность новых соединений. 

В данной задаче необходимо предсказать для каждого соединения 3 показателя:
- **IC50 (mM)** - концентрация, подавляющая 50% активности вируса
- **CC50 (mM)** - концентрация, вызывающая токсичность для 50% клеток
- **SI (Selectivity Index)** - индекс селективности

Метрика оценки: **среднее RMSE** по трем целевым переменным


## Данные

- **train.csv** - 750 объектов с известными значениями целевых переменных
- **test.csv** - 250 объектов, для которых необходимо сделать предсказания

## Запуск проекта

### Установка пакетов

Для запуска проекта необходимо установить следующие пакеты:

```pip install numpy pandas scikit-learn matplotlib seaborn optuna catboost xgboost```

### Данные

Поместите файлы train.csv и test.csv в папку data/ в корне проекта

### Запуск

Запустите все ячейки ноутбука `ML_Final.ipynb`
Ноутбук последовательно:
- загружает и анализирует данные
- выполняет предобработку данных
- генерирует новые признаки
- обучает финальную многозадачную модель
- создает файл submission_multi.csv с результатом предсказания готовым для отправки

Все случайные процессы фиксированы параметром `random_state=42` для обеспечения воспроизводимости.
